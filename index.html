Sobre Mim

Olá! Meu nome é Alex Santos. Sou estudante de Análise e Desenvolvimento de Sistemas na UNINTER e um profissional dedicado em transição de carreira para a área de tecnologia.

Minha trajetória profissional inclui uma passagem pelo Exército Brasileiro (de março de 2023 a abril de 2024), onde desenvolvi disciplina e resiliência. Atualmente, trabalho em uma agência de viagens, onde aproveito todas as oportunidades para atuar na área de TI, resolvendo problemas técnicos e otimizando processos.

Recentemente, tive a satisfação de liderar uma ação voluntária para uma ONG, a Casa Do Tiaminho, preparando 45 notebooks doados para serem usados em aulas de informática. Essa experiência prática reforçou minha paixão por usar a tecnologia para gerar impacto positivo.

Principal Projeto de Formação

Atualmente, estou focado na mentoria "Super Programador", professor Renato Gava um programa intensivo de mais de 30 horas que cobre todas as camadas do desenvolvimento de software. Abaixo, detalho as competências que estou adquirindo em cada etapa:

Passo 1: Lógica de Programação Absoluta

Fundamentos para organizar sequências de instruções e resolver problemas de forma estruturada.

Passo 2: JavaScript

Domínio da linguagem essencial para criar páginas web interativas, uma das três principais tecnologias da Web.

Passo 3: HTML e CSS

Estruturação de conteúdo com HTML e estilização de layout com CSS para a construção completa do Front-End.

Passo 4: REST API com C#

Construção do Back-End, aprendendo C# e a interface REST API para comunicação segura entre aplicações.

Passo 5: Banco de Dados

Criação e gerenciamento de coleções de informações organizadas para serem facilmente pesquisáveis e atualizáveis.

Passo 6: Git e GitHub

Utilização de sistema de controle de versão para colaboração e hospedagem de código-fonte.

Passo 7: LinkedIn para Programadores

Estratégias para otimização de perfil profissional e atração de oportunidades no mercado de TI.

Passo 8: Angular

Introdução ao framework web Angular, um passo avançado para desenvolver aplicações mais robustas e modernas.
